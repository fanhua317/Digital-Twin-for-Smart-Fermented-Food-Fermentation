# 苹果液态设计 UI 文档（夜间模式优先）

目标：将现有“工业监控大屏”风格重塑为 Apple 风格的液态玻璃视觉，夜间模式为默认并适配日间模式切换。

## 1. 设计原则

- **液态玻璃**：半透明材质、柔和高光、低对比层次，通过模糊与叠层表现深度。
- **克制而精致**：减少硬边框与强色块，用柔和渐变、阴影与微发光替代。
- **清晰可读**：夜间模式下保证文字与关键数值对比度。
- **信息分层**：卡片层级 3 档（基底 / 关键卡 / 弹出层）。

## 2. 色彩系统（夜间默认）

### 2.1 背景
- `--bg-base`: #0B0C10
- `--bg-panel`: rgba(18, 20, 26, 0.55)
- `--bg-glass`: rgba(26, 28, 36, 0.42)
- `--bg-glow`: rgba(255, 255, 255, 0.04)

### 2.2 文本
- `--text-primary`: #E7E9EE
- `--text-secondary`: #B7BCC7
- `--text-muted`: #8B92A1

### 2.3 强调色（状态色，低饱和）
- `--accent-green`: #42E07B
- `--accent-yellow`: #FFC857
- `--accent-red`: #FF6B6B
- `--accent-blue`: #5BC0FF
- `--accent-purple`: #9B8CFF

## 3. 材质与阴影

- **玻璃卡片**：
  - 背景：`--bg-glass`
  - 边框：`rgba(255,255,255,0.08)`
  - 背景模糊：`backdrop-filter: blur(16px)`
  - 内阴影：`inset 0 1px 0 rgba(255,255,255,0.08)`
  - 外阴影：`0 8px 30px rgba(0,0,0,0.35)`

- **浮层/弹窗**：提高模糊与不透明度，强调聚焦。

## 4. 排版与字号

- 字体建议：`"SF Pro Display", "SF Pro Text", "PingFang SC", "Helvetica Neue", sans-serif`
- 标题：18-22px，权重 600
- 关键数值：24-32px，权重 600
- 正文：13-15px

## 5. 布局规范

- **导航栏**：半透明玻璃侧栏，图标 + 文本，选中态使用柔光高亮条。
- **卡片布局**：保持现有网格逻辑，替换为玻璃层卡片。
- **图表区域**：使用弱网格、细线条、柔和渐变。

## 6. 组件规范

### 6.1 卡片 Card
- 取消硬边框与深色块，改为玻璃材质。
- 标题与操作区使用轻量分隔线或渐隐线。

### 6.2 按钮 Button
- 主按钮：渐变 + 轻微高光。
- 次按钮：玻璃底 + 描边。
- 危险按钮：低饱和红 + 轻发光。

### 6.3 Tag / Badge
- 背景使用透明色（如 `rgba(66,224,123,0.18)`）。
- 文本使用对应强调色。

### 6.4 表格 Table
- 行高增加至 44px。
- 行 hover 使用 `rgba(255,255,255,0.04)`。
- 表头改为半透明底 + 轻描边。

### 6.5 图表
- ECharts 主题：深色 + 细网格线 `rgba(255,255,255,0.06)`。
- 数据线使用柔和渐变与轻发光。

## 7. 动效

- 页面加载：卡片轻微上浮 + 淡入（200-300ms）。
- 数据更新：数值微闪烁或高光过渡。
- Hover：卡片阴影增强、玻璃高光增强。

## 8. 夜间模式细节

- 默认深色主题。
- 预留日间模式：背景改为 `#F7F8FB`，卡片白色玻璃，阴影变浅。
- 避免纯黑与纯白，保持柔和层次。

## 9. 适配范围

本次改造优先覆盖：
- 综合监控
- 窖池监控
- 设备管理
- 告警中心
- 生产管理

## 10. 交付方式

按页面逐步替换样式，确保功能与数据逻辑不回退。
